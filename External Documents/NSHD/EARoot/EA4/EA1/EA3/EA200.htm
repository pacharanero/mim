Constraint Type Status Detail
Goal 1. Goal Approved
Open new Call for the patient
Description 2. Description Approved
The user will open a new Call when the patient contacts the service.
Trigger 3. Trigger Approved
The patient contacts NHSD (via an 0845 call)
Post-condition - Call closed 5.Post-condition Approved
Call closed, no document created
Post-condition - Call closed and document created 5.Post-condition Approved
If the Call is closed with no further consultation required then the Call is closed and an NHSD Report is created, ready to be sent.
Post-condition - Call opened 5.Post-condition Approved
If further consultation is required with any area of the NHSD service then the Call will be left open.
Post-condition - RBAC failure 5.Post-condition Approved
If the user does not have the required RBAC then activity will end with a failure.
