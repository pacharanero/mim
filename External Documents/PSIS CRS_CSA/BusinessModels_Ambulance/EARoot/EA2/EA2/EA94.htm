Information about the pointers to specific individual Data Objects (e.g. Digital Image of ECG Trace ,or, a web-page) that are a result of this Procedure having being performed.

This group is to facilitate the inclusion of image pointers where local systems have this capability enabled.

NOTE:   Images themselves are not expected to be included due to file size

See Expected Functionality documentation
