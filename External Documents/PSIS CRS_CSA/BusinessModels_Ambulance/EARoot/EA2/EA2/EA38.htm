Details about one Pharmaceutical used in this Care Event 

Administration Event ID    
-The unique identifier for this administration event

Administration Type    
-A code-value pair to identify whether this is an administration of a medication dose or a course of medication

Route/Site of administration    
-A code to identify the route and/or site of administration used to give this pharmaceutical instance.

Administration Status    
-The status of this administration event

Time of administration    
-Time that this medication was administered

Time administration was recorded    
-Time that this medication was recorded as having been administered

Dosage of this administration    
-The quantity of substance used when:

a) it is measured as a dosage 
b) as an infusion the total amount administered

Rate of Administration    
-The rate at which the medication is administered. Only relevant if administration is an infusion

Additional Administration information    -Textual additional information about this administration.
The most common use of this would be to report changes of rate of infusion during administration
