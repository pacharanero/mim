Importing a Linked Document

This document has been accessed by 'double-clicking' on a document artefact within the model, either from the navigation page or the project browser. This document should either be removed (by the deletion of the document artefact) or overwritten by importing an RTF document.

To import a document to be linked to the document artefact, see the following instructions:

  • Right-click on this document

  • Select File -- Import…

  • Select the (RTF) document to be imported

  • Click the Open button
