 Indicates the type of a device (e.g. ECG machine). 
     Code representing a device, taken from one the following code systems: 
     
         
             SNOMED CT UK 
         
         A code from Snomed-CT (UK) may be a very general code, with device type in originalText, or may be accompanied by additional qualifying codes from the same coding system. 
         
             Read Codes V2 
         
         The codes available in the Read Codes are currently limited. However, where necessary the originalText element may be used to specify a more detailed type of device.
