 Where the EhrCompoundLevel attribute value is 'cluster', the vocabulary is used to name subdivisions that group together a logically connected set of observations and circumstances applicable to those observations. 
     A code representing the name of a cluster of statements (e.g. a series of full blood counts), taken from one of the following cod systems: 
     
         
             SNOMED CT UK 
         
         A code from Snomed-CT (UK) may be accompanied by additional qualifying codes from the same coding system. 
         
             Clinical Terms (Read Codes) v3 
         
         A code from Clinical Terms (Read codes) Version 3 may be accompanied by additional qualifying codes from the same coding system. 
         
             Read Codes Version 2 
         
         
             Read Codes Four Byte Set 
         
         
             Egton Codes 
         
         
             EMIS Drug Codes 
         
     
     An optional code from another of the coding systems described above, representing a mapping to the first code, may be used as a "translation". 
     A translation code is not required if the data was originally recorded in Snomed-CT UK; otherwise the "translation" code should have an identical or more general meaning than the original code.
