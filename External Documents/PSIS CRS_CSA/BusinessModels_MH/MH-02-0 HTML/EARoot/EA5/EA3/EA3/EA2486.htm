Attribute Details
public dt_Identifier
  SDSGPGeneralNationalCode
Notes: The GP General National Code (nhsGnc) is held against the Organisation Person entity in SDS. The NHS Data Dictionary refers to the GNC as the GENERAL MEDICAL PRACTITIONER GMP (PPD) CODE. GNC is however the commonly known name.<br /><br />The GP General National Code is created from the DIN with a leading G and a check digit added. The GNC is allocated by the PPD to a doctor when they become a member of a GP practice. This code moves with the GP from practice to practice and may only be held against the GP in one GP practice. <br /><br />A doctor may have more than one GNC where the GP is a member of more than one GP practice; A GNC will be assigned to the Practitioner for each additional practice. Tthis will be a spurious code which is not based on the DIN.
