Constraint Type Status Detail
Goal 1. Goal Approved
Send document to PSIS and other recipients
Description 2. Description Approved
The user will choose to send a document which has been completed on the local application. The document will be sent to PSIS and other recipients.
Trigger 3. Trigger Approved
Local document created.
Pre-condition 1 4. Pre-condition Approved
The document must have been finalised
Pre-condition 2 4. Pre-condition Approved
An LR must have been established between the user and the patient in a previous activity
Pre-condition 3 4. Pre-condition Approved
The patient must have a valid NHS Number (Patient ID) obtained from PDS.
Post-condition - Document Not Sent (error) 5.Post-condition Approved
The activity may end with the document not being sent, because of a failure in the process.
Post-condition - Document Sent 5.Post-condition Approved
The activity ends with the document being sent.
Post-condition - RBAC Failure 5.Post-condition Approved
If the user does not have the required RBAC then activity will end with a failure.
Post-condition - Document Not Sent (User Choice) 5.Post-condition Approved
The activity may end with the document not being sent, because the user choose not to.
Post-condition - Consent Failure (Other Orgs) 5.Post-condition Approved
The process ends because the individual does not consent for their information to be shared with other organistaions.
