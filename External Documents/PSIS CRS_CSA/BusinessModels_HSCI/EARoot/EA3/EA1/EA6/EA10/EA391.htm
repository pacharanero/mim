Where a table structure is used to represent the overview assessment, the system may use a single text section for the whole assessment, or each area of assessment.

If an additional author is required, this requires a new text section, and hence a new table.

Suppliers should consider and consult with user groups as to whether it is necessary to render the author of a text section (if different to document author), and if so then consideration should be given to how this should appear within the document.
