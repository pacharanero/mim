The system must supply a careSetting on the assessmentEvent template - a code from the vocab CDACareSettingTypeSnCT .

The system should use one of the location choices available on the assessmentEvent template to give additional details of the actual location.

Where careSetting is mandatory on the chosen location template, the system should apply an appropriate rule on how to populate
For example:
- always use same careSettingType as used on assessmentEvent template, or
- only allow logical combinations of care setting types, e.g. more specialised location at this level - 22232009 hospital , then 224885007 Hospital consulting room.

This applies to:
- Contact Assessment document
- Overview Assessment document
- Specialist assessment Outcome document
