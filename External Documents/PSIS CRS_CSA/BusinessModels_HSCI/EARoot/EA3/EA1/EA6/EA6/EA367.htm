The format of many assessments in use is:
Question + multiple choices of answers + free text supporting the chosen answer.

In preparing the Information Examples and Example MIM messages, practitioners indicated to NHS CFH that it is often not necessary for the assessment output document to contain all the options that were available to the assessor, just the chosen option and supporting text.

Suppliers should consider and consult with user groups as to whether it is necessary to render all the options of multiple choice question (sometimes the range of options gives context to the chosen option), and if so then consideration should be given to how this should appear within the document.

This applies to:
- Contact Assessment document
- Overview Assessment document
- Specialist assessment Outcome document
