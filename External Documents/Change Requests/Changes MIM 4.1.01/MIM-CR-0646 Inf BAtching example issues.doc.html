Short Title of Issue

 

Batching examples in the MIM (4.1) carry sender/recipient detail in both batch and transmission wrappers.

 

 

Description

 

The PDS batching examples in MIM 4.1 carry sender recipient info in both the batch and transmission wrappers. This is incorrect. There should be no duplication.

 

 

Perceived Impact

 

No impact

 

Backward Compatibility Status

No issues – correction

Reason for issue being raised

 

Issue raised internally.

 

Suggested Action

 

There are issues round batching schemas and therefore examples. We may not publish schemas that can be used so should we be publishing examples. If we do publish examples then the existing examples may change and there should be additional examples.

 

CTT Resolution and Current Status

 

The existing PDS schemas need to be corrected.

Depending on what is the outcome of the schema issue there will be a decision made as to whether examples will be removed or more will be created. My gut feeling is that examples are useful.
