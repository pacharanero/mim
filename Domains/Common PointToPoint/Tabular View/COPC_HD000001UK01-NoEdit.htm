PayloadInformation
A class that holds information about what is being carried within the message .

Indicates this is an event
[1..1] [M] code (CV {CNE:P2PActType } )
A code from the vocabulary shown indicating the type of payload information act.

The Coded Plain flavour of Coded Value (CV) datatype shall be used to carry this information as follows:

  • The code attribute shall contain the value
  • The codeSystem attribute shall contain the OID for the vocabulary concerned
  • The displayName attribute shall contain the text associated with the code
[1..1] [M] id (II)
This is a unique identifier for this instance of an observation of a payload information act.

The Identifier Global datatype flavour is used to carry the first identifiers follows:

  • The root attribute shall contain a DCE UUID.
[1..1] [M] messageType (II)

This attribute holds the message type.

The IdentifierExternal datatype flavour is used to carry this information as follows:

  • The root attribute shall contain the OID "2.16.840.1.113883.2.1.3.2.4.18.17"
  • The extension attribute shall contain the Message Type ID as allocated by NHS CfH
