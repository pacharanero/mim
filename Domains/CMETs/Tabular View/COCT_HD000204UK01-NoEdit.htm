{ Fixed="AP" }

Specifies how the author contributes to the context of the VPI and whether it may be propagated to descendent acts whose association allows such propagation.

"A" (Additive)- the association adds to any context of the same or more specific type that has been conducted from the parent act.

"P" (Propagating)- the association is conducted to all child acts where the ActRelationship.contextConductionInd is set to "true" - where not stated this defaults to "true".
