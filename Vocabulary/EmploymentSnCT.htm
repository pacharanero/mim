Vocabulary for EmploymentSnCT

Code System Id

2.16.840.1.113883.2.1.3.2.4.15

Version: 1.0
Status: active
Date: 2007-11-20

Description

A subset of SNOMED-CT intended for use in the UK, a set of codes which represent aspects of employment including occupation and employment status. This vocabulary uses distinct sub-sets of codes.

Any code from the SNOMED CT UK EmploymentStatus subset with subset original id 6921000000134.

Any code from the SNOMED CT UK Occupation subset with subset original id 6661000000133.

The use of this subset is Advisory as defined in latest version of "NPFIT-ELIBR-AREL-P1R2-0178 Technical Guidance for Implementation of Templated CDA Domains."
