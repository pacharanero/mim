Vocabulary for PhysicalWellBeingSnCT

Code System Id

2.16.840.1.113883.2.1.3.2.4.15

Version: 1.0
Status: active
Date: 2007-11-29

Description

A subset of SNOMED-CT intended for use in the UK, a set of codes which represent different aspects of a persons well being, including areas such as pain, sleeping, breathing and disabilities conditions. This vocab contains multiple distinct sub-sets of codes. 

The subsets are as follows:

Any code from the SNOMED CT UK Pain subset with subset original id 7661000000130.

Any code from the SNOMED CT UK Sleep subset with subset original id 7671000000137.

Any code from the SNOMED CT UK Swallowing and Oral Health subset with subset original id 7681000000135. 

Any code from the SNOMED CT UK GeneralPhysicalWellBeing subset with subset original id 7691000000138.  

Any code from the SNOMED CT UK Breathing subset with subset original id 7701000000138.

Any code from the SNOMED CT UK Communication subset with subset original id 7711000000136. 

Any code from the SNOMED CT UK Consciousness and Seizures subset with subset original id 7721000000133.  

Any code from the SNOMED CT UK Skin Care and Wound Management subset with subset original id 7731000000130.   

The use of this subset is Advisory as defined in latest version of "NPFIT-ELIBR-AREL-P1R2-0178 Technical Guidance for Implementation of Templated CDA Domains."
