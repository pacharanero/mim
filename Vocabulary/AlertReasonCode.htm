Vocabulary for AlertReasonCode

Code System Id

2.16.840.1.113883.2.1.3.2.4.17.194

Version: 4.0
Status: active
Date: 2008-03-18

Description

A generic vocabulary acting as a place holder for a number of specific codes which identify the alert reason codes.  The codes may come from definitions found in the MIM or may be defined externally.

LegitimateRelationships (OID = 2.16.840.1.113883.2.1.3.2.4.17.244 )
The concepts identifying alert reason codes in Legitimate Relationship Alerts are defined externally in FileCM and can be retrieved using the reference: NPFIT-FNT-TO-REQ-DEL-0162.02 "National Alerting of Local LRs: Requirements Specification"

Sealed Envelopes (OID = 2.16.840.1.113883.2.1.3.2.4.17.200 )
The concepts identifying alert reasons in Sealed Envelope Alerts are defined externally by the Sealed Envelope Team in the Sealed Envelopes Supplier Requirements Documentation, Appendix D.

Dissent (OID = 2.16.840.1.113883.2.1.3.2.4.17.60 )
The concepts identifying alert reasons for dissent are defined in the MIM vocabulary DissentOveride.
