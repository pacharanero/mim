Vocabulary for MentalEmotionalWellBeingSnCT

Code System Id

2.16.840.1.113883.2.1.3.2.4.15

Version: 1.0
Status: active
Date: 2007-11-20

Description

A subset of SNOMED-CT intended for use in the UK, a set of codes which represent different aspects of personal emotional wellbeing, including mood, behaviour and impaired cognitive functioning. This vocab contains multiple distinct sub-sets of codes.

The list of subsets are as follows:
Any code from the SNOMED CT UK Mood subset with subset original id 7741000000135.

Any code from the SNOMED CT UK Behaviour subset with subset original id 7751000000137.

Any code from the SNOMED CT UK Cognition subset with subset original id 7761000000139.

The use of this subset is Advisory as defined in latest version of "NPFIT-ELIBR-AREL-P1R2-0178 Technical Guidance for Implementation of Templated CDA Domains."
