Vocabulary for CDADocumentSectionType

Code System Id

2.16.840.1.113883.2.1.3.2.4.17.223

Version: 3.0
Status: active
Date: 2007-12-20

Description

A set of codes to identify the document section type.

Individual domains may specify the codes to use as follows:

  • For HSCI Overview Assessment Document codes from the vocabulary CAFInformationSet (OID=2.16.840.1.113883.2.1.3.2.4.17.222)should be used.
  • For Diagnostic Imaging Report codes from the vocabulary DICDADocumentSectionType (OID=2.16.840.1.113883.2.1.3.2.4.17.243) should be used.
  • For Integrated Care and Support Plan document codes from the vocabulary TextHeadingType (OID=2.16.840.1.113883.2.1.3.2.4.17.225) should be used.
